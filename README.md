# أدعية إسلامية (Flask)

A clean and modern Islamic web app inspired by Islambook-style sections, built with Flask.

## Features

- Quran page with searchable verses.
- Hadith page with book-based filtering.
- Azkar page (morning, evening, after salah).
- Prayer times page by city/country (Aladhan API).
- Responsive modern interface (Bootstrap + custom CSS).

## Quick Start

1. Create and activate a virtual environment.
2. Install dependencies:

   pip install -r requirements.txt

3. Run app:

   python app.py

4. Open in browser:

   http://127.0.0.1:5000

## Notes

- Content is structured and curated for demonstration and learning.
- You can expand data in [data/content.py](data/content.py).
